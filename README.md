
# Culqi Reto




## Run Locally

Clone the project

```bash
  git clone https://github.com/RenzoAlcala/token-api-serverless.git
```

Go to the project directory

```bash
  cd token-api-serverless
```

Install dependencies

```bash
  npm install
```

Install Typescript and Jest globally

```bash
  npm install -g typescript 
  npm install -g jest
```


Start the server

```bash
  npm run start
```
